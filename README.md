Para instalar aplicacion:

- Tener java 17
- maven v3+

Modo local:

ejecute: 

- mnv clean install

- java -jar users-0.0.1-SNAPSHOT.jar &


Modo docker:

- docker-compose up -d



Abrir en navegador 

- http://localhost:8080/api/users
